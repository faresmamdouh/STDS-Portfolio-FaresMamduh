# Task 01: Used Cars Insights Dashboard (Excel)

## Task Overview
This task focuses on analyzing and visualizing data from the used cars market in Egypt using Microsoft Excel.  
The goal is to transform raw data into meaningful insights through dashboards, KPIs, and charts.

## Objectives
- Analyze used car prices, mileage, models, and features
- Build interactive dashboards using Excel
- Summarize key market trends and comparisons
- Present insights in a clear and visual format

## Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- KPI Cards
- Data Cleaning & Aggregation

## Dataset
The dataset contains used car listings in Egypt, including:
- Price
- Mileage
- Brand & model
- City
- Car features (A/C, automatic transmission, etc.)
- Manufacturing year
